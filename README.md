# Student Grade Calculator

## Description

This Python program calculates a student's grade based on marks entered by the user. It validates input, assigns grades using if-elif-else statements, and displays encouraging messages.

## Grading Logic

* A: 90-100
* B: 80-89
* C: 70-79
* D: 60-69
* F: 0-59

## Features

* Uses functions
* Uses if-elif-else statements
* Validates marks between 0 and 100
* Uses while loop for invalid inputs
* Displays encouraging messages
* Handles non-numeric input

## How to Run

```bash
python student_grade_calculator.py
```
